Privacy Policy

Privacy Policy

Effective Date: December 25, 2025

Welcome to the AxelBase Bidirectional Text Inspector — a completely client-side, open-source web tool designed to detect and visualize Unicode bidirectional control characters in any text you paste.

We collect absolutely no personal data. Ever.

No Data Collection Whatsoever

This tool runs 100% in your browser. When you paste or type text into the inspector:

  • No information is sent to any server
  • No IP addresses are logged
  • No cookies, localStorage, or tracking pixels are used
  • No analytics services (Google Analytics, Plausible, etc.) are loaded
  • Your clipboard content never leaves your device

How Your Text Is Processed

All analysis happens instantly and locally using pure JavaScript. The moment you paste suspicious text, the browser scans it character-by-character looking for the twelve known bidirectional control characters. Results are displayed immediately — and disappear the moment you close or refresh the page.

Third-Party Services

We use only the Bootstrap 5 CSS and JS files from the official CDN for styling. These files do not track users or set cookies when loaded with Subresource Integrity (SRI) hashes. No other external services are used.

GitHub Pages Hosting

The site is statically hosted on GitHub Pages. GitHub itself may log basic server access for security, but this is outside our control and contains no personal data from you.

Contact

Questions? Open an issue on our public GitHub repository.

Your privacy is not a feature — it’s the foundation. By using this inspector, you can be certain that what you paste stays yours alone.
